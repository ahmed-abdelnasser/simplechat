## About application

Simple realtime chat app room laravel , vue with pusher
## run application

- clone repositry
```bash
git clone 
```
- install composer 
```bash
composer install
```
- copy env file 
```bash 
cp .env.example .env
```
- generate new key 
 ```bash 
php artisan key:generate
```
- modify database config at .env

 ```bash 
npm 
```
