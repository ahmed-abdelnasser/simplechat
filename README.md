# About application

Simple realtime chat app room laravel , vue with pusher
# run application

1. clone repositry

2. install composer 
```bash
composer install
```
3. copy env file 
```bash 
cp .env.example .env
```
4. generate new key 
 ```bash 
php artisan key:generate
```
5. modify database config at .env

6. run migrate
 ```bash 
php artisan migrate 
```

### enjoy :)
